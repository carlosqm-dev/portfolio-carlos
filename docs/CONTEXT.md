# Qué es este proyecto

Portafolio personal de desarrollador. Sitio web con una página principal estática que enlaza a páginas de detalle por cada proyecto mostrado.

## Objetivo

Conseguir oportunidades de trabajo o freelance como desarrollador Full Stack.

## Propuesta de valor

Desarrollador Full Stack enfocado en diseño de sistemas escalables, buenas prácticas de arquitectura, y uso de IA como potenciador del flujo de trabajo (no como reemplazo del criterio técnico).

## Tono

Cercano y directo. Nada de lenguaje corporativo o excesivamente formal. El portafolio debe sonar a una persona real, no a un template genérico.

## Estructura

- Página principal (estática): presentación, propuesta de valor, grid de proyectos.
- Página de detalle por proyecto.
